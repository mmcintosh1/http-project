The Beginning of HTTP

HTTP was created by Tim Berners-Lee in 1991 while working at CERN. Its purpose was to allow computers to communicate and share documents across the newly created World Wide Web.

The first version, HTTP/0.9, only supported simple GET requests and returned HTML documents. As the web grew, HTTP continued to evolve to support more features and better performance.

Timeline

Version Year Main Improvement
HTTP/0.9 1991 Basic document retrieval
HTTP/1.0 1996 Headers and status codes
HTTP/1.1 1997 Persistent connections
HTTP/2 2015 Multiplexing and faster loading
HTTP/3 2022 Uses QUIC for improved speed and reliability
