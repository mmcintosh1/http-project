Important HTTP Concepts

  • HTTP: The protocol used to transfer web pages and other resources.
  • HTTPS: The secure version of HTTP that encrypts data.
  • URL: The web address used to locate a resource.
  • Request: A message sent by a browser asking for information.
  • Response: The information returned by the web server.
  • Status Code: A number that tells whether the request succeeded.
  • Web Server: A computer that stores and delivers websites.
